# Gólyaest

![Gólyaest](/golyaest.png)

Péntek délutánra egy izgalmas csapatépítő programmal készülünk a gólyák számára! Az este folyamán sok érdekes játékokban vehetnek részt, ahol fejleszthetik csapatmunka és kommunikációs készségeik. Sok szórakoztató feladattal készülünk, emellett olyan interaktív játékokkal amelyek során megtudhatnak többet egymásról és az iskoláról.

## Információk

- **Időpont:** 2023. szeptember 15. 17:00-21:00
- **Helyszín:** Tornacsarnok
